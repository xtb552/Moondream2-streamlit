Moondream2 Vision 模型 Streamlit 应用
423240616
这是一款Streamlit应用，使用Moondream2视觉模型，基于上传的图片和用户提供的提示生成文本。

特色
上传PNG或JPEG格式的图片。
输入提示词来指导文本生成。
根据上传的图片和提示生成文本。
如何运行
安装所需的Python软件包：
pip install -r requirements.txt
运行Streamlit应用：
streamlit run vision.py
在你的网页浏览器中打开该应用。http://localhost:8501
用途
用文件上传器上传图片。
在文本输入栏输入提示。
点击“生成”按钮，根据图片和提示生成文本。
关于模型
Moondream1视觉模型体积小巧但强大的视觉模型，性能优于其体积的两倍模型。
